# Generador de Paletas de Colores PI Moldulo 1 Henry

## Descripción
Este proyecto consiste en una aplicación web que genera paletas de colores aleatorias. El usuario puede elegir la cantidad de colores que desea visualizar y generar una nueva combinación con solo presionar un botón.
Cada color se muestra junto con su código HEX para facilitar su uso en proyectos de diseño o desarrollo web.
---
## Funcionalidades
- Generación de paletas de colores aleatorias.
- Selección de 6, 8 o 9 colores.
- Visualización del código HEX de cada color.
- Botón para generar una nueva paleta.
- Diseño responsive para dispositivos desktop y móviles.
- HTML semántico y consideraciones básicas de accesibilidad.
